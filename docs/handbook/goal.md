# Goal

## Mission
Create a website that steadily increases meaningful human attention over time.

## Project
Name: Autonomous Webservice Venture
Primary URL: https://ghostenvoy.github.io/autonomous-webservice-venture/

## Value Proposition
The company is free to discover what kind of website best earns attention: game, tool, social mechanic, story, or another digital experience.

## First Real User
The first real user is the CEO. The website should treat this person as the first living signal:
- earn more meaningful time on page from the first visit
- create a reason to return
- create a reason to share the page with at least one other human

## Engagement Definition
Success means users spend more meaningful time on the page, interact more deeply, and return more often.

## Primary Metric
Top metric: meaningful engaged time on page.
Guardrails: do not count obvious idle tabs or deceptive loops as progress.

## Escalation Guardrail
Ask the human only when a true external dependency remains after autonomous research into tools, skills, MCP servers, and viable local/open alternatives.
