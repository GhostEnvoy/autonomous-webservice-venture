# Strategy

## Current Strategic Thesis
Engagement will improve if the website gives the first visitor a strong immediate hook, a short interactive loop that rewards curiosity, and an earned reason to share the page with someone else.

## Active Strategic Bets
1. Treat the CEO as the first live signal and optimize for a stronger first-session experience before chasing broader traffic.
2. Add one compact interactive loop that rewards exploration within the first 60 seconds.
3. Earn the first share through curiosity and delight, not through spammy begging or fake urgency.
4. Instrument real engagement before optimizing distribution.
4. Exhaust autonomous options before escalating: research tools, skills, MCP servers, and viable local/open alternatives first.

## Known Unknowns
- What experience format best increases meaningful time on page for a first-time visitor?
- Which interactions correlate with genuine desire to return or share?
