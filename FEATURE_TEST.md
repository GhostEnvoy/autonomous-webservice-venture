# webservice-venture

A landing page to collect user pain points with waitlist-like lead capture (email, use_case, pain, optional willingness to pay). Uses local SQLite storage and provides a stats endpoint for demand analysis.

## Quick Start
```bash
cd web
npm install
npm start
```
Open: http://127.0.0.1:4040

## Features
- Lead capture form with email, use case, pain point, and optional willingness to pay
- Local SQLite storage for collected signals
- Stats endpoint for demand analysis
- Analytics instrumentation for engaged time tracking
